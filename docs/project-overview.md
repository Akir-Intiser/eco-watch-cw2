# ECO-WATCH CW2 Project Overview

## Project Name
ECO-WATCH

## Module
COM682 Cloud Native Development

## Project Description
ECO-WATCH is a cloud-native environmental sightings web application. It allows users to upload environmental sightings with an image, location, category, and description. The system stores image files in cloud storage and stores sighting metadata in a cloud database.

## Main Features
- Upload a new environmental sighting
- View all submitted sightings
- Edit an existing sighting
- Delete an existing sighting
- Store uploaded images separately from metadata
- Use REST API endpoints for CRUD operations
- Prepare the project for Azure deployment
- Include evidence for coursework video demonstration

## Planned Azure Services
- Azure Static Web Apps or Azure CDN for frontend hosting
- Azure Functions or Logic Apps for REST API endpoints
- Azure Blob Storage for image uploads
- Azure Cosmos DB for sighting metadata
- Azure SQL Database for user account data
- Azure Application Insights for monitoring
- GitHub Actions for CI/CD deployment
